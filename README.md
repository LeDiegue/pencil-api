# API PENCIL


#### Dependencies
##### Rails
```
gem install rails
```
#### Run Run
First, you need install dependencies. Please, run the following command:
```
bundle install
```
and then, you need run the migrations
```
rake db:migration
```
and to finish, run:
```
rails s
```

Ready! :D.

You can check in localhost on port 3000 by default. [localhost:3000](http://localhost:3000/)
